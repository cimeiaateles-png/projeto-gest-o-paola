# 💄 Gestão Paola Estética: Controle Financeiro Inteligente

Desenvolvi esta aplicação para solucionar um problema real de gestão financeira, unindo lógica de programação a uma interface elegante e funcional.

**Nota de Evolução e Transparência**: Este projeto está em desenvolvimento ativo. Como sou bem detalhista, já planejei futuras atualizações tanto no refinamento visual quanto na precisão técnica dos dados e valores, que serão revisados para refletir a realidade do mercado.

---

## Inteligência de Dados & Curadoria
Diferente de sistemas genéricos, a base de cálculos deste projeto foi construída através de um trabalho minucioso de levantamento de dados:

**Tratamento em Planilha**: Todos os valores de insumos e preços de serviços foram inicialmente estruturados e calculados em uma planilha de gestão, onde defini as margens de lucro e os custos fixos por procedimento.
**Migração para Código**: Esses dados foram extraídos e convertidos em objetos JSON dentro do JavaScript, permitindo que a aplicação realize os cálculos de forma dinâmica e automatizada.
**Trabalho Técnico**: Esse processo garantiu que cada centavo de abatimento de insumo e cada repasse de 30% estivesse alinhado com a realidade financeira da clínica.

---

## Tecnologias Utilizadas

Para garantir um sistema leve e profissional, utilizei:

**HTML5 Semântico**: Estruturação focada em acessibilidade.
**CSS3 Customizado**: Flexbox e Grid para um design responsivo e premium.
**JavaScript (Vanilla)**: Lógica de negócios e manipulação do DOM.
**LocalStorage**: Armazenamento de dados no navegador, oferecendo total autonomia à usuária.

---

## Desafios Superados & Funcionalidades

Minha atenção aos detalhes me levou a criar soluções personalizadas:

1.  **Seletor Customizado**: Componente de busca com rolagem interna que não quebra o design do cabeçalho.
2.  **Automação Financeira**: Cálculo automático de repasse (30%) e abatimento de insumos em tempo real.
3.  **Interface Clean**: Experiência focada na produtividade da profissional de estética.
